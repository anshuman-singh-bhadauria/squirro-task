# squirro-task
Create a web app.

## Installation instructions
- git clone 
- cd squirro-task
- conda env create -f conda-env.yml
- conda activate sq-app
- python init_db.py
- python app.py (Launches a server on 0.0.0.0:30003)
- to test the service run from another terminal
- conda activate sq-app
- python test.py
